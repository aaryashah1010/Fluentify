# 📌 Epic & Sprint Breakdown with User Stories

---

## Epic 1 (Sprint 1: Documentation, DB Setup & Authentication)

### User Stories
- **USL1: Learner Registration & Profile**  
  - **Front:** As a learner, I want to register with my details and goals, so that I get a personalized learning path.  
  - **Back:** Given I provide valid details and goals with proficiency level, When I submit the form, Then my profile should be created and a recommended learning path shown, or else we have an error.

- **Added: Login & Authentication System**  
  - **Front:** As a learner/admin, I want a secure login system, so that I can safely access the platform.  
  - **Back:** Given I enter valid credentials, When I log in, Then I should be authenticated securely, or else receive an error message.
